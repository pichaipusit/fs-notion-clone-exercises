## Setting up project
Setup .env file
 ```javascript
# Deployment used by `npx convex dev`
CONVEX_DEPLOYMENT=
NEXT_PUBLIC_CONVEX_URL=

NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=

EDGE_STORE_ACCESS_KEY=
EDGE_STORE_SECRET_KEY=
```
Install
 ```javascript
npm install
```
Setup Convex
 ```javascript
npx convex dev
```
Start the app
 ```javascript
npm run dev
```
